title: MAC OS X 编辑快捷键
---

MAC 没有 Windows 上的 `Home`, `End`等按键，但是 MAC 默认支持 Emacs 的一些快捷键，可以直接使用。

###移动

`⌃ + F` 前进一个字符

`⌃ + B` 后退一个字符

`⌃ + E` 行末

`⌃ + A` 行首

`⌃ + N` 下一行

`⌃ + P` 上一行

`⌃ + V` 移动一个屏幕

###删除

`⌃ + H` 向后删除一个字符

`⌃ + D` 向后删除一个字符

`⌃ + K` 从光标位置删除到行末

###粘贴

`⌃  + Y` 粘贴剪贴板中的内容
