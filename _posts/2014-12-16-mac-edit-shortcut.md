MAC 没有 Windows 上的 `Home`, `End`等按键，但是 MAC 默认支持 Emacs 的一些快捷键，可以直接使用。

### 移动

`⌃ + A` 行首

`⌃ + E` 行末

`⌃ + F` 前进一个字符

`⌃ + B` 后退一个字符

`^ + L` 将光标或所选内容置于可见区域中央

`⌃ + P` 上一行

`⌃ + N` 下一行

`⌃ + V` 移动一个屏幕

`^ + O` 在插入点后插入一行

`^ + T` 将插入点后面的字符与插入点前面的字符交换

### 删除

`⌃ + H` 向后删除一个字符

`⌃ + D` 向后删除一个字符

`⌃ + K` 从光标位置删除到行末

### 粘贴

`⌃ + Y` 粘贴剪贴板中的内容

### 插入

`^ + o` 插入一个新行
